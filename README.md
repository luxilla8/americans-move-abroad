# Americans Move Abroad

Directory website helping Americans relocate overseas — 20 country profiles with visa pathways, cost of living, healthcare, and expat community data.

Built for midlife professionals (40+) considering an international move.

## Features

- 20 researched country profiles
- Interactive filtering by region, budget, and visa ease
- Full detail modals with 9 data sections per country
- Learn More links to official visa portals, expat guides, and US embassies
- Dark mode support
- Responsive design

## Tech

Single-page static site — HTML + CSS + vanilla JS. No build step required.

## Deploy

Drop `index.html` on any static host (Vercel, Netlify, GitHub Pages, etc.).
